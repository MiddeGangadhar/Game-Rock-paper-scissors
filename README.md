**Rock–Paper–Scissors Game (Python)**

This is a simple Rock–Paper–Scissors game written in Python.
The user chooses Rock (0), Paper (1), or Scissors (2), and the computer randomly selects one.
Then the program shows who wins.

How the Game Works

0 = Rock

1 = Paper

2 = Scissors

Rules:

Rock beats Scissors

Paper beats Rock

Scissors beats Paper

If both choose the same thing → Draw

Features

Simple user input

Random computer choice

Shows what user and computer selected

Tells if you win, lose, or draw

Beginner-friendly Python code

How to Run the Program

Install Python on your computer

Save the code in a file named game.py

Open Command Prompt (Windows) or Terminal (Mac/Linux)

Run this command:

python game.py

Example Output
0 = Rock
1 = Paper
2 = Scissors
Enter 0 for Rock, 1 for Paper, or 2 for Scissors: 1
You chose: Paper
Computer chose: Rock
You win!
